## Clonar el repositorio

**https**

```bash
git clone --recurse-submodules https://github.com/FAMAF-resources/4to_2C-Logica-FAMAF.git
```

**ssh**

```bash

```

## Update

```bash
git pull && git submodule update --init --recursive
```
